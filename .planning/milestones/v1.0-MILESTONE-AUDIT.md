## E2E Flow

**CLI Pipeline (the only user flow):**
1. `python main.py -p <id> [-w <days>]` — parses args
2. Loads Statcast + Pitching+ data, filters to pitcher
3. Classifies appearances as SP/RP
4. Computes all engine metrics (fastball, arsenal, execution, workload)
5. Assembles PitcherContext Pydantic model
6. Renders `to_prompt()` (~544 tokens)
7. Streams scout-voice narrative from Claude to terminal

**Error paths verified:**
- Invalid pitcher ID → exit 1 with message
- Missing args → exit 2 with usage
- Missing ANTHROPIC_API_KEY → exit 1 with message
