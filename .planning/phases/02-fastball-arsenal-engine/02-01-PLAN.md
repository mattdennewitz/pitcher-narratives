  <action>
    Create engine.py and tests/test_engine.py following RED-GREEN-REFACTOR.

    **engine.py structure:**

    1. Module docstring explaining purpose: computation engine transforming PitcherData into pre-computed analysis with qualitative trend strings.

    2. Imports: `from __future__ import annotations`, `from dataclasses import dataclass`, `import polars as pl`, `from data import PitcherData`.

    3. `__all__` list with public API: `compute_fastball_summary`, `compute_velocity_arc`, `FastballSummary`, `VelocityArc`.

    4. Constants:
       - `_FASTBALL_TYPES = frozenset({"FF", "SI", "FC"})` -- standard Statcast fastball classification
       - `_VELO_THRESHOLD = 0.5` -- mph below which velocity delta is "Steady" (per user decision)
       - `_PPLUS_THRESHOLD = 5` -- points below which P+/S+/L+ delta is "Steady"
       - `_SHARP_VELO_THRESHOLD = 2.0` -- mph above which delta is "sharply" (per research recommendation)
       - `_SHARP_PPLUS_THRESHOLD = 10` -- points above which P+ delta is "sharply"
       - `_MIN_PITCHES = 10` -- minimum pitches for per-type analysis (per user decision); below this flag small_sample=True

    5. Delta string helpers (private, used by all engine functions):
       - `_velo_delta_string(delta: float, threshold: float = _VELO_THRESHOLD) -> str`:
         abs(delta) < threshold -> `f"Steady ({delta:+.1f})"`
         abs(delta) >= _SHARP_VELO_THRESHOLD -> `f"{direction} sharply ({delta:+.1f} mph)"`
         else -> `f"{direction} {abs(delta):.1f} mph"`
       - `_pplus_delta_string(delta: float, threshold: float = _PPLUS_THRESHOLD) -> str`:
         abs(delta) < threshold -> `f"Steady ({delta:+.0f})"`
         abs(delta) >= _SHARP_PPLUS_THRESHOLD -> `f"{direction} sharply ({delta:+.0f} points)"`
         else -> `f"{direction} {abs(delta):.0f} points"`
       - `_movement_delta_string(delta: float) -> str`:
         Always include direction and magnitude in inches: `f"{direction} {abs(delta):.1f} in"`
         abs(delta) < 0.5 -> `f"Steady ({delta:+.1f} in)"`

    6. `_identify_primary_fastball(pitch_type_baseline: pl.DataFrame) -> str | None`:
       Filter pitch_type_baseline to rows where pitch_type is in _FASTBALL_TYPES.
       Sort by n_pitches descending, return the first pitch_type. Return None if no fastball types.

    7. `_get_window_game_dates(data: PitcherData) -> list`:
       Extract unique game_date values from data.window_appearances as a list. Used to filter Statcast and P+ CSVs to window.

    8. `_is_cold_start(data: PitcherData) -> bool`:
       Return True when len(window_appearances) >= len(appearances) -- window covers full season.

    9. `_weighted_window_pplus(appearance_type_df: pl.DataFrame, window_dates: list, pitch_type: str) -> dict`:
       Filter pitcher_type_appearance CSV to window_dates and pitch_type.
       Compute n_pitches-weighted average of P+, S+, L+ columns.
       Return dict with keys "P+", "S+", "L+", "n_pitches".
       Handle empty result (return dict with None values).

    10. Dataclasses:

    ```python
    @dataclass
    class FastballSummary:
        """Pre-computed fastball quality analysis ready for LLM."""
        pitch_type: str            # e.g., "FC"
        pitch_name: str            # e.g., "Cutter"
        season_velo: float
        window_velo: float
        velo_delta: str            # e.g., "Down 1.2 mph"
        season_p_plus: float
        window_p_plus: float | None
        p_plus_delta: str
        season_s_plus: float
        window_s_plus: float | None
        s_plus_delta: str
        season_l_plus: float
        window_l_plus: float | None
        l_plus_delta: str
        season_pfx_x: float
        window_pfx_x: float
        pfx_x_delta: str
        season_pfx_z: float
        window_pfx_z: float
        pfx_z_delta: str
        small_sample: bool
        cold_start: bool

    @dataclass
    class VelocityArc:
        """Within-game velocity progression for most recent appearance."""
        game_pk: int
        game_date: str             # ISO date string
        available: bool
        early_velo: float | None
        late_velo: float | None
        drop: float | None         # late - early (negative = dropped)
        drop_string: str           # e.g., "Dropped 1.3 mph" or "Single inning -- no velocity arc available"
        innings_pitched: int
    ```

    11. `compute_fastball_summary(data: PitcherData) -> FastballSummary | None`:
        - Call _identify_primary_fastball on data.pitch_type_baseline. Return None if no fastball.
        - Get window_dates from _get_window_game_dates.
        - Check cold_start via _is_cold_start.
        - **Velocity:** Filter data.statcast to primary fastball type. Compute season mean release_speed. Filter to window_dates, compute window mean. Generate delta string (or cold start string).
        - **P+/S+/L+:** Get season values from data.pitch_type_baseline for the primary fastball type. Get window values via _weighted_window_pplus from agg_csvs["pitcher_type_appearance"]. Compute deltas.
        - **Movement:** Compute season mean pfx_x/pfx_z from full statcast. Compute window mean from window-filtered statcast. Generate delta strings.
        - **Small sample:** True if window fastball pitch count < _MIN_PITCHES.
        - **Pitch name:** Get human-readable name from statcast: `statcast.filter(pitch_type==primary).select("pitch_name").unique()[0]`.
        - If cold_start, set all delta strings to "Full season in window -- no trend comparison".

    12. `compute_velocity_arc(data: PitcherData, fastball_type: str) -> VelocityArc`:
        - Find most recent appearance: data.appearances sorted by game_date desc, take first row's game_pk and game_date.
        - Filter statcast to that game_pk and fastball_type.
        - Get unique innings sorted. If len(innings) < 2, return VelocityArc with available=False, note string.
        - Otherwise: early_innings = innings[:2], late_innings = innings[-2:]. Compute early_velo and late_velo means. drop = late_velo - early_velo.
        - Generate drop_string: abs(drop) < 0.5 -> "Held steady", drop < 0 -> f"Dropped {abs(drop):.1f} mph", drop > 0 -> f"Gained {drop:.1f} mph".
