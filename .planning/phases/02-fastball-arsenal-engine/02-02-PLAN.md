    6. `compute_arsenal_summary(data: PitcherData) -> list[PitchTypeSummary]`:
       - Get all pitch types from pitch_type_baseline (sorted by n_pitches descending).
       - Get window_dates, cold_start flag.
       - For each pitch type:
         - **Usage:** Count pitches of this type in full statcast / total pitches * 100 = season_usage_pct. Same for window-filtered statcast = window_usage_pct. Compute usage_delta via _usage_delta_string.
         - **P+/S+/L+:** Season values from pitch_type_baseline. Window values via _weighted_window_pplus from agg_csvs["pitcher_type_appearance"]. Compute deltas via _pplus_delta_string.
         - **Pitch name:** From statcast `pitch_name` column for matching pitch_type. Use `.unique()` and take first.
         - **Small sample:** n_pitches in window < _MIN_PITCHES.
         - **Single pitch type:** If only 1 type, usage is 100%/100%, set usage_delta to "Only pitch type".
         - If cold_start, set all deltas to "Full season in window -- no trend comparison".
       - Return list sorted by season_usage_pct descending.

    7. `compute_platoon_mix(data: PitcherData) -> PlatoonMix`:
       - Get window_dates, cold_start.
       - Build pitch_type-to-name mapping from statcast.
       - Compute platoon baseline via _compute_platoon_baseline on agg_csvs["pitcher_type_platoon"].
       - For each pitch_type in pitch_type_baseline (by usage):
         For each platoon_side in ["same", "opposite"]:
           - **Season usage:** From statcast, filter to pitch_type, compute what % of pitches to this platoon_side are this type. Use _stand_to_platoon(stand, data.throws) to map stand to platoon.
           - **Window usage:** Same but filtered to window_dates.
           - **Season P+:** From platoon baseline.
           - **Window P+:** From agg_csvs["pitcher_type_platoon_appearance"] filtered to window_dates, weighted average.
           - If no pitches exist for this (pitch_type, platoon_side) combo, set available=False with note "Not thrown to {platoon_side}-side batters".
           - Compute deltas.
       - Return PlatoonMix with all splits.

    8. `compute_first_pitch_weaponry(data: PitcherData) -> FirstPitchWeaponry`:
       - Filter statcast to pitch_number == 1 (first pitch of each at-bat, per user decision).
       - Season: count by pitch_type, divide by total first pitches = season_pct per type.
       - Window: filter first pitches to window_dates, compute window_pct per type.
       - For each pitch type, compute delta via _usage_delta_string.
       - Handle cold_start.
       - Order entries by window_pct descending.
       - Return FirstPitchWeaponry with totals.
