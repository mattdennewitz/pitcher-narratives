       ```python
       class PitcherContext(BaseModel):
           model_config = ConfigDict(arbitrary_types_allowed=True)

           pitcher_name: str
           pitcher_id: int
           throws: str
           role: str                                   # "SP" or "RP" (most recent appearance)

           fastball: FastballSummary | None
           velocity_arc: VelocityArc | None
           arsenal: list[PitchTypeSummary]              # Top 4 only
           platoon_mix: PlatoonMix
           first_pitch: FirstPitchWeaponry
           execution: list[ExecutionMetrics]            # Top 4 only
           workload: WorkloadContext

           def to_prompt(self) -> str:
               """Render as prompt-ready markdown under 2,000 tokens."""
               ...
       ```

    6. `to_prompt()` implementation -- per user decision: markdown with headers, bullet points, and small tables. Per Claude's discretion: exact formatting, section ordering, missing data handling.

       **Section ordering** (Claude's discretion):
       ```
       # {pitcher_name} ({throws}HP) — Scouting Context

       ## Role
       - Most recent: {role}
       - {workload summary: appearances, consecutive days concern}

       ## Primary Fastball: {fastball.pitch_name} ({fastball.pitch_type})
       - Velo: {season_velo:.1f} season / {window_velo:.1f} recent — {velo_delta}
       - Stuff+ (P+): {season_p_plus:.0f} season / {window_p_plus:.0f} recent — {p_plus_delta}
       - Movement: H {pfx_x_delta}, V {pfx_z_delta}
       - Velocity arc (last outing): {drop_string}

       ## Arsenal
       | Pitch | Usage | Delta | P+ | Delta |
       |-------|-------|-------|----|-------|
       (top 4 rows)

       ## Execution
       | Pitch | CSW% | Zone% | Chase% | xRV100 pctl |
       |-------|------|-------|--------|-------------|
       (top 4 rows)

       ## Platoon Shifts
       Notable shifts as bullet points (skip unavailable splits)

       ## First-Pitch Tendencies
       Top 3 first-pitch types as bullet points

       ## Recent Appearances
       | Date | IP | Pitches | Rest |
       (window appearances, most recent first)
       ```

       **Missing data handling** (Claude's discretion):
       - If fastball is None: skip the fastball section, add "No standard fastball identified" note
       - If xrv100_percentile is None: show "—" in table
       - If xwhiff_p/xswing_p is None: show "—"
       - If platoon split available=False: skip that split
       - Never render the literal string "None"

       **Token budget**: Keep top 4 pitch types in arsenal and execution tables. Use abbreviated headers. Skip verbose delta strings in tables (just show the numeric delta or short form). Estimate with `len(output) / 4` -- research found ~392 tokens for a comprehensive format, well within 2,000.

       - Determine role from most recent appearance: `data.appearances.sort("game_date", descending=True).row(0, named=True)["role"]`
       - Return PitcherContext with all fields populated.
