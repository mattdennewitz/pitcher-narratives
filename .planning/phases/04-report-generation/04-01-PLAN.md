    1. Module-level Agent:
       ```python
       from pydantic_ai import Agent
       from pydantic_ai.settings import ModelSettings
       agent = Agent(
           'anthropic:claude-sonnet-4-6',
           output_type=str,
           system_prompt=_SYSTEM_PROMPT,
           model_settings=ModelSettings(max_tokens=4096),
       )
       ```

    2. _SYSTEM_PROMPT constant (multi-line string):
       - Persona: "You are a veteran MLB pitching analyst writing a scouting report"
       - Anti-recitation (per locked decision): "Write insight, not stat lines. Reference numbers to support observations, don't list them. If a delta is small, say so and move on."
       - Instruction: write prose paragraphs, use data tables only where they genuinely aid comprehension
       - Instruction: never fabricate trends or claims not present in the provided context data
       - Report length guidance: aim for 3-6 paragraphs depending on how much is noteworthy

    3. Role-conditional guidance constants:
       - _SP_GUIDANCE: emphasize pitch mix depth, stamina indicators (velocity arc, late-inning command), evolution across recent starts
       - _RP_GUIDANCE: emphasize workload patterns (consecutive days, rest impact), short-window weapon deployment, leverage readiness

    4. _build_user_message(ctx: PitcherContext) -> str:
       - Select _SP_GUIDANCE or _RP_GUIDANCE based on ctx.role
       - Combine role guidance + ctx.to_prompt() into a single user message string
       - Pattern: f"## Analysis Focus\n{guidance}\n\n## Pitcher Data\n{ctx.to_prompt()}"

    5. generate_report_streaming(ctx: PitcherContext, *, _model_override=None) -> str:
       - Build user message via _build_user_message(ctx)
       - Call agent.run_stream_sync(user_message, model=_model_override) if override provided, else agent.run_stream_sync(user_message)
       - Iterate stream.stream_text(delta=True), print each delta to stdout (end='', flush=True)
       - Collect chunks, print final newline, return joined string
       - IMPORTANT: run_stream_sync is NOT a context manager -- call directly, no `with` statement

    6. __all__ = ["generate_report_streaming"]
